# Switch to Neon Database

## Why Neon?
- ✅ IPv4 compatible (works from AWS)
- ✅ Free tier: 3GB
- ✅ PostgreSQL (same as Supabase)
- ✅ Prisma compatible
- ✅ Serverless (auto-scales)

## Steps:

1. Go to https://neon.tech
2. Sign up / Sign in
3. Create new project "embpay"
4. Copy connection string
5. Update .env

Neon provides direct IPv4 connections that work from AWS without issues.
